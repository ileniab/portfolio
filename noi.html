Web design & data journalism project

Noi, internet-dipendenti

Interactive web design and data journalism article
in collaboration with Sole24Ore Lab

objective

With the world under quarantine, this project aimed to design multimedia and interactive storytelling, in collaboration with Il Sole 24 Ore, to narrate the impact on our digital habits, and the resulting increase in bandwidth consumption. For the development of the project, we combined web design, data visualization, motion-graphic, and front-end development.

WHAT IS

WHAT IS Noi internet-dependenti is a data journalism project that combines storytelling, data visualization and interactive elements, enhancing a "user-centered" exploration of the article. In fact, through the digital narrative path, is possible to discover, for instance, the average data consumption of the user himself, thanks to a small beginning questionnaire. The data of the user are compared with the world average consumption and, eventually,  it's shown how much it would be consumed on average, if everyone had the very same user’s activity.
