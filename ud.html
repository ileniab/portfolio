INTERACTIVE INSTALLATION

UD particellar

university indoor installation, san marino

This project was also commissioned by the University of San Marino.
It is an interactive installation built with touchdesigner. This visual project is composed by a particle system that interacts with body movements via kinect camera. It was designed for the university's open days.

technologies

  • Kinect xbox one
  • Projector
  • Touchdesigner

TECHNICAL PROCESSES

Information collected by the kinect:
Silhouette of the user's body.
To this silhouette I applied a force that moves the particle elements.
